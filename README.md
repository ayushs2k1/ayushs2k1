### About Me

I'm a software engineer and researcher interested in distributed systems, AI/ML, and building things that scale. Currently pursuing my MS in CS at NYU (GPA 3.9) while contributing to [Ceph](https://ceph.io/) as a Google Summer of Code 2026 participant.

Previously, I spent 2.5 years at **Hewlett Packard Enterprise** working on a carrier-grade media server serving 100M+ users, where I also co-invented and patented an on-device LLM inference pipeline for real-time scam detection (presented at **Mobile World Congress 2024**). Before that, I interned at **Samsung R&D** building low-level networking libraries in C++.

Most recently I was a **ML Research Intern at NYU's CILVR Lab** under Prof. Pavel Izmailov, studying transformer scaling laws and building an SVG generation model.

- 🏆 Hackathon wins: HackNYU, Startup Week Buildathon, Daytona HackSprint, ElevenLabs Worldwide Hack, Pulse NYC
- 📜 Certifications: Certified Kubernetes Application Developer (CKAD), AWS Certified Cloud Practitioner
- 📄 Published: [A Comparative Study on Efficiencies of Variants of CNNs on Image Classification](https://www.researchgate.net/publication/381516909_A_Comparative_Study_on_Efficiencies_of_Variants_of_Convolutional_Neural_Networks_based_on_Image_Classification_Task)
